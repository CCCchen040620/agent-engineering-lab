## 阶段术语卡：Python 后端与 RAG 入门

- API：应用程序接口，前端或其他系统调用后端功能的入口。
- Endpoint：接口地址，例如 `/api/v1/chat`。
- Request：请求，客户端发给后端的数据。
- Response：响应，后端返回给客户端的数据。
- Query Parameter：查询参数，例如 `?file_type=pdf`。
- Path Parameter：路径参数，例如 `/documents/1` 里的 `1`。
- Status Code：HTTP 状态码，例如 200、201、404、409、422。
- Model：数据模型，用来描述数据结构。
- Validation：校验，检查数据是否符合要求。
- Service：业务逻辑层，负责真正处理功能。
- Router：路由层，负责接收请求并调用 service。
- Citation：引用来源，说明回答依据来自哪段资料。
- RAG：检索增强生成，先检索资料，再生成回答。
- Refusal：拒答，知识库没有资料时明确说明无法回答。
- Test Isolation：测试隔离，让测试不依赖真实可变数据。
