# 英文术语卡

## 第 1 周

| 英文 | 中文 | 记忆方式 |
| --- | --- | --- |
| Python | Python 编程语言 | 我们使用的主要语言 |
| code | 代码 | 给计算机的指令 |
| program | 程序 | 一组可以运行的代码 |
| terminal | 终端 | 用文字向电脑下命令的窗口 |
| command | 命令 | 在终端里输入的操作 |
| file | 文件 | 保存代码或数据的容器 |
| folder | 文件夹 | 用来组织文件 |
| path | 路径 | 文件在电脑中的地址 |
| error | 错误 | 程序无法按预期运行的信息 |
| output | 输出 | 程序运行后显示的结果 |
